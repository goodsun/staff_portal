# proposals — 提案置き場

labo-portal への機能提案・アイデアをここに置く。  
AIも人間も同じフローで提案できる。

---

## 提案フロー

```
1. template.md をコピーして NNN_提案名.md を作る
2. draft/proposals/ にコミット
3. PRを出す（タイトル: "proposal: 提案名"）
4. レビュー・議論
5. マージ = 「検討中」に昇格
6. 実装決定 → docs/design.md に移動・Phaseに追加
```

## ルール

- 誰でも提案できる（メフィ・みぃちゃん・テディ・アリス・マスター）
- 提案はMarkdownで書く
- 却下も記録として残す（削除しない）
- 「なぜ必要か」を必ず書く

## ファイル命名

```
NNN_plugin_name.md   例: 001_cco_dashboard.md
```
